# 🎨 Pablo Rafael — Portfolio Designer

Bem-vindo ao repositório do meu portfólio profissional. Este espaço foi desenvolvido para apresentar minha trajetória, competências técnicas e os principais projetos autorais e comerciais criados como Designer Gráfico, Especialista em UI/UX e Analista de Marketing.

O objetivo principal desta aplicação é centralizar meus trabalhos de forma visualmente impactante, limpa e responsiva, proporcionando uma excelente experiência de navegação tanto para clientes quanto para recrutadores.

---

## 💻 Sobre o Projeto

O portfólio foi planejado seguindo conceitos sólidos de **Design Thinking**, tipografia marcante e alto contraste. Ele atua como uma vitrine interativa das minhas principais frentes de atuação:

*   **UI/UX & Digital Product Design:** Criação de interfaces centradas no usuário, prototipagem avançada e fluxos de navegação otimizados.
*   **Design Gráfico & Identidade Visual:** Criação de logotipos, conceitos visuais estruturados, campanhas institucionais e peças publicitárias.
*   **Marketing Digital & Performance:** Criação de layouts voltados para conversão (Landing Pages), estratégias visuais para e-commerce e peças direcionadas a campanhas de tráfego.

---

## ✨ Funcionalidades e Diferenciais

*   **Design Totalmente Responsivo:** Adaptação fluida e diagramação simétrica para dispositivos móveis, tablets e desktops (Mobile-First adaptado).
*   **Navegação Inteligente:** Menus retráteis dinâmicos para otimização de espaço na tela e transições suaves de scroll entre seções.
*   **Seções Estruturadas:** Áreas dedicadas para apresentação pessoal, galeria de projetos selecionados, competências técnicas e canais diretos de conversão.
*   **Performance Otimizada:** Carregamento rápido de imagens e assets visuais mantendo a alta fidelidade das cores e detalhes.

---

## 🛠️ Tecnologias Utilizadas

Para garantir um código leve, limpo e de excelente desempenho, a arquitetura do projeto utiliza tecnologias web nativas (Vanilla Web Techs):

*   **HTML5:** Estruturação semântica de layouts e foco em acessibilidade.
*   **CSS3 Avançado:** Flexbox para alinhamento e distribuição da grade de projetos, efeitos modernos de hover/transição e Media Queries dedicadas.
*   **JavaScript (Vanilla):** Lógica dinâmica para interatividade da interface e manipulação de classes para navegação mobile.

🚀 Como Visualizar Localmente
O projeto é estático e não necessita de instaladores ou servidores pesados.

Clone o repositório em sua máquina:
```
Bash
git clone [https://github.com/pablo-rafael/portfolio-designer.git](https://github.com/pablo-rafael/portfolio-designer.git)
```

Entre no diretório do projeto:
```
Bash
cd portfolio-designer
Abra o arquivo index.html diretamente em seu navegador web de preferência.
```
